    <h1 class="animated">Tutorial for Download</h1>

    <h3>1. Click on Garry's Mod -> Properties -> Betas</h3>
    <h3>2. Install x86-64-Chromium + 64-bit binaries version</h3>
    <h3>3. Download the theme files, open the Garrysmod/garrysmod folder, and extract the lua and resource folders there. Be sure to click "Replace" in the window that appears.</h3>
    <h3>4. All set! If you get bored with the theme, you can simply type "gbox_switch" into the console, and the theme will revert to the default design.</h3>
